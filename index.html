Project 01

HeartLink


Heartlink is a product designed to bring people together. Wether you're far, far away or you live in the same city, the person you care the most about is right at your fingertip! Put your finger on the sensor, the device sends your pulse through a server to the device you've connected to through the mobile app and if the other person does the same, the connections count goes higher! This is a simple prototype and proof of concept and is not largely available. Features:

  • Live pulse detection
  • Large distance communication through a TCP server using the built in WI-FI
  • Bluetooth communication for configuration through the mobile app
  • Data aquisition (from the pulse sensor)

Microprocessor ESP32 Embedded Data aquisition and processing C++ Python TCP server/client architecture
